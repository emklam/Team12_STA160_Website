# Team 12, STA 160
## Predicting Housing Prices in California Website

Leverage machine learning to predict housing and Airbnb listing prices across California. This project features an interactive map, predictive models, and dashboards to help users explore trends and make data‑driven decisions.

Access our site here : https://emklam.github.io/Team12_STA160_Website/index.html 

 ### Overview

California Housing Price Predictor is a class project that combines geospatial visualization with machine learning to estimate property prices and analyze Airbnb and housing market statistics across California. The site includes:

• Housing Price Prediction

• Housing Map Statistics

• Airbnb Map Statistics

• Housing Trends Dashboard

### Features

• Interactive map with replace "tooltips, filters, and region-level aggregation"

• ML-powered price predictions for housing and Airbnb listings

• Housing trends dashboard 

### Architecture

• Frontend: (HTML/CSS/JS)

• Visualization: (OpenStreetMap)

• ML: (scikit-learn/XGBoost/PyTorch)

• Data: Public datasets (e.g., California housing data, Airbnb listings, county crime rate, walkability, etc.)

• Hosting: GitHub Pages (gh-pages)
